AGUI is not currently production ready. It is even not in alpha state.

In general: it is possible to create single window application for Mac OS with some limitations.

API is not stabilized and may be changed on any commit. There is no documentation. There is only one driver - for Mac OS. Not all base functionality have been implemented in Drivers and GUI. Only limited number of Controls have been implemented.
